# Usage

```
python3 main.py
```

You can specify the following options interactively:

- the size of the puzzle (N * M)
- the initial board (you can also let the program generate an initial board that is reachable randomly)
- the algorithm (BFS or A* search)

If the puzzle you give is unreachable, the program terminates with RuntimeError.

If the puzzle is reachable, the program shows the number of searched boards and how to solve the puzzle.

# Example

```shell
> python3 main.py
Enter the number of rows (N):
3
Enter the number of columns(M):
3
Do you specify the initial board? (y)es / (n)o:
y
Enter the initial board:
1 2 3
4 0 5
6 7 8
Choose the algorithm. (b)fs / (a)*:
a
```
